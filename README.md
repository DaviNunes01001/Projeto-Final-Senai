
# 📌 Sobre o Projeto

Este é um projeto do curso de **Análise e Desenvolvimento de Sistemas** para demonstrar a habilidade em desenvolvimento web do grupo.  
O objetivo é criar um site simples e funcional para demonstrar o conhecimento e as habilidades dos integrantes.

---

## 🎯 Objetivo do projeto

O objetivo deste projeto é desenvolver um site **Front-End** utilizando:

- React
- Vite
- JavaScript

O site consome uma API desenvolvida pelos integrantes do grupo em aulas anteriores.

---

## 🛠️ Tecnologias utilizadas

- React
- React Router
- JavaScript
- CSS
- Vite
- Fetch API
